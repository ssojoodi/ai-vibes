University of Waterloo has the best in class coop program.

One of the pillars on which the program is built is the employers commitment to providing structured feedback to the coop students during and at the end of the coop term.  These forms were paper-based back in my day, then became PDFs and now a simple webpage for filling out the forms.

However, to collect this information, save it and work with it inside of an organization, we either have to pass annotated PDFs back and forth or turn the form into text.

So as a solution to this collaboration challenge, I'm creating this no-backend utility form to help capture the information, save it as draft in local storage, and export/import as JSON for sharing internally. There's also an Export as MD feature which will help with filling out the official form in the Waterloo systems.
