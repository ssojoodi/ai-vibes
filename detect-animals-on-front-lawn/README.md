# Single-shot prompt with Claude Sonnet 4

```
Problem: dogs pee on my front lawn and ruin the grass.

I need to design a python script which as input takes the video feed from a webcam and detects if a dog or any animal is on my lawn. I need to be able to create a sensitivity zone in the webcam feed, using a visual editor. You can assume the webcam’s picture frame doesn’t move. As soon as an object of a certain size is detected in the image frame, the script is to print a message. This message needs to be printed as long as the object is in the picture. You can sample image frames at once per second.
```
